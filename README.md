# Obsidian Folder Locator

一个快速定位文件夹的Obsidian插件。通过快捷键快速搜索并定位文件夹。

## 特性

- 使用快捷键（默认`F3`）快速打开文件夹搜索
- 智能展开父文件夹
- 高亮显示定位路径
- 支持在设置中自定义快捷键

## 使用方法

1. 按下快捷键（默认`F3`）打开搜索框
2. 输入文件夹名称
3. 选择目标文件夹
4. 自动定位并高亮显示

## 安装

1. 下载 `main.js` 和 `manifest.json`
2. 将这两个文件放入你的 vault 的 `.obsidian/plugins/folder-locator/` 目录
3. 重启 Obsidian
4. 在设置中启用插件
5. 可在设置中自定义快捷键

## 开发
```bash
# 安装依赖
npm install

# 开发模式
npm run dev

# 构建
npm run build
```
